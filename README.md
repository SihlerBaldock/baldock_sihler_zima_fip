# Interactive Media Design - Level 2 - Final Integrated Project
This webpage will include the homepage, products page, and about page of the revived ZIMA drink brand. The page must be full responsive through mobile, tablet, and desktop, and it must contain some dynamically added content using arrays and objects with Javascript.

## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open index.html in the browser of your choice.
